# 实机信息提供清单

## 1. 目的

当前代码已经实现屠夫召唤、战斗、宝箱拾取、背包检测、回城、仓库存放和返回循环，但视觉阈值、屏幕坐标和等待时间仍是占位配置。

需要在安装了《暗黑破坏神 IV》的电脑上采集以下信息，才能进行准确校准和实机验证。

请不要提供账号、密码、令牌或其他敏感信息。

## 2. 信息优先级

| 优先级 | 内容 | 用途 |
|---|---|---|
| 必需 | 游戏显示设置 | 确定截图尺寸和坐标基准 |
| 必需 | Boss 房完整截图 | 校准召唤装置、角落和宝箱 |
| 必需 | 有怪、无怪截图 | 校准敌人检测 |
| 必需 | 空、半满、全满背包截图 | 校准背包占用率 |
| 必需 | 城镇落点、仓库、传送门截图 | 校准回城和仓库流程 |
| 必需 | 实际交互和仓库快捷键 | 防止脚本按错键 |
| 必需 | 各阶段实际耗时 | 设置移动和加载等待时间 |
| 建议 | 技能可用和冷却截图 | 后续实现真实技能 CD 检测 |
| 建议 | 不同品质掉落截图 | 改善掉落识别 |
| 建议 | 死亡、断线等异常截图 | 后续增加异常恢复 |
